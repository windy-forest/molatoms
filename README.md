# molatoms — GAMESS 出力から 3D プリント用の分割ファンデルワールスモデルを作る

GAMESS の計算結果（最適化構造）から分子のファンデルワールス（vdW）空間充填モデルを作り、
**原子ごとに分割した STL ファイル**を生成するツール群です。各ピースを 3D プリントして
組み立てると、分子模型が完成します。組み立てを助けるための識別子と二面角合わせの
切り欠きを、切断面に刻印できます。

処理は 2 段階です。

```
benzene.out ──(1) gamess_to_json.py──▶ benzene.json ──(2) vdw_to_stl.py──▶ benzene_stl/*.stl
 GAMESS出力                              原子座標+半径                        原子ごとのSTL
```

---

## 必要環境

- Python 3.9 以上
- 段階 (1) `gamess_to_json.py` … 標準ライブラリのみ（追加インストール不要）
- 段階 (2) `vdw_to_stl.py` … 下記をインストール

```bash
pip install -r requirements.txt
```

## 使い方

### (1) GAMESS 出力 → JSON

GAMESS のジオメトリ最適化出力から平衡構造の原子座標を抽出し、JSON に変換します。
各原子の元素記号・原子番号・座標(Å)に加え、後段で球の半径として使う
**ファンデルワールス半径**を付与します。

```bash
python gamess_to_json.py benzene.out
# -> benzene.json を出力
```

| オプション | 説明 |
| --- | --- |
| `-o, --output` | 出力 JSON 名（既定: 入力名の拡張子を `.json` に） |
| `--geometry {equilibrium,last,first}` | 抽出する構造。既定 `equilibrium`（`EQUILIBRIUM GEOMETRY LOCATED` 直後の座標）。`last`/`first` で最後/最初のステップ |
| `--stdout` | ファイルに書かず標準出力へ |

出力 JSON の例:

```json
{
  "source": "benzene.out",
  "geometry": "EQUILIBRIUM GEOMETRY",
  "units": "angstrom",
  "atom_count": 12,
  "atoms": [
    { "index": 0, "element": "C", "atomic_number": 6,
      "x": 0.0036, "y": -0.0000, "z": 0.0001, "vdw_radius": 1.7 }
  ]
}
```

### (2) JSON → 分割 STL

vdW 球を原子ごとに分割し、切断面に組み立て用の刻印を付けて STL を出力します。

```bash
python vdw_to_stl.py benzene.json --combined
# -> benzene_stl/ に benzene_00_C.stl ... と確認用 benzene_combined.stl を出力
```

| オプション | 説明 |
| --- | --- |
| `-o, --output-dir` | STL 出力先（既定: `<入力名>_stl`） |
| `--scale` | Å→mm のスケール係数（既定 `10.0`。例: C の vdW 半径 1.7Å→17mm 球） |
| `--subdivisions` | 球メッシュの細かさ（既定 `3`。滑らかにしたいなら 4〜5） |
| `--combined` | 全原子を結合した確認用 STL も出力 |
| `--no-marks` | 識別子・切り欠きを付けない（分割のみ） |

## 分割のしくみ

各原子は「原子核を中心とする vdW 半径の球」です。球が重なる相手ごとに、
2 球の交線が乗る平面（radical plane）で球を切り、相手側の出っ張りを削ります。
平面位置は半径差を反映し、原子 A の中心からの距離は

```
dA = (d² + rA² − rB²) / (2d)        d = 中心間距離
```

となります（半径が等しければ中点、異なれば中点になりません）。

- **重なる全ペア**を切るので空間が隙間なく分割され、各ピースは互いに貫通せず
  ぴたりと組み合います（パワー・ダイアグラム分割）。
- 各原子を**独立に**処理し、平面は常に「その原子の球」にしか作用しないため、
  ベンゼン中心のように多数の球が重なっても二重カットや破綻が起きません。
- ブール演算は `manifold3d` を用い、出力メッシュは watertight になります。

## 切断面の刻印（組み立てガイド）

水素を含まない化学結合（C–C, C–N など）の切断面には、結合を一意に識別し、
二面角を合わせるための刻印が付きます（C–H 結合や非結合の接触面は無印）。
各結合に `0〜255` の ID を割り当て、次のように符号化します。

| 部位 | 担当ビット | 意味 |
| --- | --- | --- |
| **4 ビット凹みドット** | 上位ニブル (0–15) | 4 個のビット穴（有=1, MSB 左）。左端に開始マーカー、下側に上下識別ライン |
| **大小 2 本の切り欠き** | 下位ニブル (0–15) | リムの溝。大=二面角の基準、小=`値×22.5°` の位置 |

4 ビット × 16 角度 = **256 結合**まで一意に識別できます。

**組み立て方:** radical plane の性質上、結合する 2 原子の切断面は同一平面・同一の交円を
共有します。

1. 両ピースの切断面のドット模様（ID）が一致するものどうしを合わせる。
2. 横から見て、両ピースの**大小の切り欠きが揃う向き**まで回す。
   これで二面角が一意に決まります（大きい溝が基準）。
3. 面どうしを接着する。
